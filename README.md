### 🛠 Prerequisites

- ☕ JDK 8 or higher
- ⚙️ Gradle or compatible build tool

### 🏗️ Build & Run

1. 🌀 Clone the repository:
   ```sh
   git clone <repository-url>
   cd QissasAlMustaghfireen
   ```
2. 🛠️ Build the project:
   ```sh
   ./gradlew build
   ```
3. ▶️ Run the application:
   ```sh
   ./gradlew run
   ```
